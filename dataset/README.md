# Dataset

Place the IMDb dataset file here:

```
IMDB Dataset.csv
```

The CSV is not included in the repository. Download it separately and add it to this folder before running:

```bash
python main.py --train
```

Expected columns: `review`, `sentiment` (`positive` / `negative`).
Dataset URL: https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews?utm_source=chatgpt.com
